# Asset Management App

## Overview
The Asset Management App is a web application designed to help users manage their assets efficiently. It provides a user-friendly interface for viewing, adding, and updating asset information.

## Project Structure
```
asset-management-app
├── src
│   ├── index.html        # Main HTML document for the application
│   ├── css
│   │   └── styles.css    # CSS styles for the application
│   ├── js
│   │   ├── app.js        # Main JavaScript file for application functionality
│   │   ├── assets.js     # JavaScript file for asset management functionalities
│   │   └── utils.js      # Utility functions for the application
│   └── assets
│       └── data.json     # Static data for the application
├── .gitignore            # Files and directories to be ignored by version control
└── README.md             # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open the `src/index.html` file in your web browser to view the application.

## Features
- **Company Info**: View information about the company.
- **Plans**: Access various investment plans.
- **Mission & Vision**: Learn about the company's mission and vision.
- **Login**: User authentication for accessing personal asset information.
- **Register**: Create a new account to start managing assets.

## Usage Guidelines
- Use the navigation buttons to explore different sections of the application.
- Follow the prompts for login and registration to access personalized features.
- Manage your assets through the provided functionalities in the app.
